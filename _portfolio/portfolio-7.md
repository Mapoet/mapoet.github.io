2013 年 6 月至 2014 年 8 月，于中南大学参与"惠州现代测绘基准服务平台"项目核心功能研发。项目承接惠州市国土局需求，旨在建立坐标基准转换综合平台，实现各部门、各县区坐标与高程成果的统一切换。

**项目目标**

在城市数据框架平台内，基于坐标转换插件实现多种数据格式图件与数据成果的相互转换，满足测绘精度标准要求。

**主要贡献**

- 使用 C++ 及 COM 技术，实现符合测绘精度标准的多坐标系转换算法
- 同时支持 C# C/S 平台与 JavaScript B/S 平台的插件调用
- 基于 SQL Server 设计坐标数据与转换参数数据库，集成至双平台
- 参与软件界面设计，完成各模块功能测试

**技术栈**

C++、COM、C#、JavaScript、SQL Server
