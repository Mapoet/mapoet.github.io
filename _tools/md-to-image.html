--- title: "Markdown 转图片与 PDF" date: 2025-06-12 permalink: /tools/md-to-image/ excerpt: "在浏览器中将 Markdown 渲染为矢量 PDF 或 PNG，支持 GFM 表格、KaTeX 公式与 Mermaid 图表" tags: - Markdown - Writing - Visualization ---

在浏览器中将 Markdown 渲染为矢量 PDF（可选中复制文字）或高清 PNG，支持 GFM 表格、KaTeX 公式与 Mermaid 图表。全部在本地完成，不上传内容。

导出 PDF：点击「导出矢量 PDF」，在打印对话框选择「另存为 PDF」并勾选「背景图形」。也可直接访问 独立页面 （适合全屏使用）。
