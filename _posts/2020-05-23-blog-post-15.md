# 基于Namelist的Fortran2018学习

```fortran
integer*4 a,b
/namelist/ab/a,b/
open(12,file='ab.namelist')
read(12,nml=ab)
write(*,fmt="(2(x,I2.2))")a,b
close(12)
```

```
/&
a=1,
b=2
/
```
